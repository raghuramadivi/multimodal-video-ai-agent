# 🎥 Multimodal Video AI Agent

An end-to-end agentic AI application that lets you upload any video 
and ask natural language questions about its content. Powered by 
Google Gemini 3.1 Pro Preview and the Phidata agentic framework, 
with real-time web search via DuckDuckGo.

## 🚀 Features
- 📤 Upload mp4 / mov / avi videos (up to 200MB)
- 🤖 Gemini 3.1 Pro multimodal video analysis
- 🔍 DuckDuckGo web search for extra context
- 💬 Natural language Q&A about video content
- 🖥️ Clean Streamlit web interface

## 🛠️ Tech Stack
- Phidata (Agentic AI Framework)
- Google Gemini 3.1 Pro Preview
- Streamlit (UI)
- DuckDuckGo Search Tool
- Python + dotenv

## ⚙️ Setup
1. Clone the repo
2. pip install -r requirements.txt
3. Add GOOGLE_API_KEY to .env file
4. streamlit run app.py
